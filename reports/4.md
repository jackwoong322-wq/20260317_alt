# 이슈 #4 — 예측 라벨(EH/EL) 표기 및 주석 박스 정보 개선

## 수정 내용

### 1. 예측 라벨 prefix 변경
- 고점: `H9` → `EH9`
- 저점: `L9` → `EL9`

### 2. 달러 가격 표시 추가
- 라벨: `EH9 82.3%($88.5k)` 형식
- 툴팁 고점/저점에도 동일 적용
- 계산: `peak_price * rate / 100` (peak_price 없으면 미표시)

### 3. Reason 항목 제거
- 툴팁에서 Reason 행 삭제

### 4. 예측 주석 색상 조정 (시각적 우선순위)
- 실선 주석: 선명한 원색 유지
- 예측 주석: 같은 계열이지만 연하게 (파스텔)
  - Hi Bear: `#ff4466` → `#FF9AB0`
  - Hi Bull: `#FFB800` → `#FFD980`
  - Lo: `#00ff88` → `#7AFFC2`

## 사양서 판단
**유지** — 기존 기능 개선, 사양 구조 변경 없음
